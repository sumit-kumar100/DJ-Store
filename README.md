Django Ecommerce Website
This is a fully-featured ecommerce website built with Django, Python, HTML, CSS, and Tailwind CSS. The website includes various functionalities commonly found in an ecommerce platform, such as a home page, product listing page with filters, product details page with multiple color and size options, checkout page, order tracking, and more.
